# data/samples/

**Local-only folder. Nothing here is committed to the public repo.**

Place your scorebook images or PDFs here for OCR experimentation. Actual files (`.jpg` / `.jpeg` / `.png` / `.heic` / `.pdf`) are gitignored because they typically contain player names, jersey numbers, and other personal information.

If you later want to share a sanitized sample publicly, create `data/samples-public/` (not gitignored) and place a redacted copy there — but by default, keep samples fully local.
